# 📝 Todo App

一個使用 Python 開發的命令列 Todo List 應用程式。

## ✨ 功能

- ✅ 查看待辦事項
- ✅ 新增待辦事項
- ✅ 刪除待辦事項
- ✅ 修改待辦事項
- ✅ 檔案儲存（todo.txt）

---

## 🛠️ 使用技術

- Python 3.11
- Function
- List
- While Loop
- File I/O

---

## 🚀 執行方式

```bash
python todo.py
```

---

## 📂 專案結構

```
todo-app/
```
